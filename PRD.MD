# 🧠 Projeto Revisão! - Sistema de Revisão Ativa

## 1. Visão Geral
O "Revisão!" é um Web App para ajudar estudantes a organizarem suas revisões, focando em vencer a curva do esquecimento e reduzir a ansiedade (medo) de falhar nos estudos.

## 2. Público-Alvo
Estudantes que precisam de um método visual para saber o que revisar e quando.

## 3. Funcionalidades (MVP)
1. **Cadastro de Estudo:** Registro de Matéria, Dificuldade, O que entendeu, e Data.
2. **Listagem de Cards:** Visualização clara dos estudos realizados.
3. **Indicador de Status:** Diferenciação visual para matérias com "Alto Medo/Dificuldade".
4. **Persistência de Dados:** Uso de JSON-Server para salvar as informações.

## 4. Tecnologias
- React + TypeScript (Vite)
- Axios (Comunicação com API)
- JSON-Server (Banco de Dados Simulado)
- Tailwind CSS (Estilização)
