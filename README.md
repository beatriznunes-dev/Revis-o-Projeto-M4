Revisão! é uma aplicação web focada em auxiliar estudantes a organizarem suas revisões, para ajudarem a memorizarem o que foi estudado, intuitiva e totalmente acessível.

🚀 Tecnologias Utilizadas
React + TypeScript: Construção de interface robusta e tipada.

Vite: Ferramenta de build ultra-rápida.

Axios: Consumo de API para persistência de dados.

CSS3 (Glassmorphism): Estilização moderna com efeitos de transparência e desfoque.

JSON Server / MockAPI: Simulação de backend para armazenamento de usuários e estudos.

Acessibilidade:

Semântica HTML5: Uso correto de tags como <main>, <aside>, <section> e <article> para facilitar a leitura por tecnologias assistivas.

Contraste de Cores: Paleta de cores testada para garantir legibilidade entre texto e fundo, atendendo a usuários com baixa visão.

🛠️ Como Executar o Projeto:

git clone https://github.com/beatriznunes-dev/Lembrei-m4

npm install

npx json-server --watch db.json --port 3001

npm run dev
